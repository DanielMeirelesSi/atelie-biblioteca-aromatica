# Ateliê Biblioteca Aromática

Site institucional e catálogo virtual desenvolvido para o **Ateliê Biblioteca Aromática**, com foco na apresentação dos produtos e no contato direto com clientes pelo WhatsApp.

O projeto possui uma interface responsiva, desenvolvida com prioridade para dispositivos móveis, permitindo que os visitantes conheçam os produtos, filtrem o catálogo e solicitem informações de maneira simples.

## Funcionalidades

* Catálogo organizado por categorias
* Busca e filtros de produtos
* Páginas individuais para cada produto
* Integração com WhatsApp por mensagens personalizadas
* Seção institucional sobre o ateliê
* Área para encomendas personalizadas
* Layout responsivo para celulares, tablets e computadores
* Metadados e estrutura voltados para compartilhamento e mecanismos de busca

## Tecnologias

* Next.js
* React
* TypeScript
* Tailwind CSS

## Executando o projeto

Certifique-se de possuir o Node.js instalado.

Instale as dependências:

```bash
npm install
```

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

Acesse o projeto em:

```text
http://localhost:3000
```

Para gerar a versão de produção:

```bash
npm run build
```
